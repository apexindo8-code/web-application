    <script>
      let records = JSON.parse(localStorage.getItem("healthRecords")) || [];

      function saveStorage() {
        localStorage.setItem("healthRecords", JSON.stringify(records));
      }

      function addRecord() {
        let name = document.getElementById("name").value;
        let age = document.getElementById("age").value;
        let location = document.getElementById("location").value;
        let condition = document.getElementById("condition").value;

        if (name == "" || age == "" || location == "" || condition == "") {
          alert("Please fill all fields");
          return;
        }

        let record = { name, age, location, condition };

        records.push(record);

        saveStorage();

        alert("Worker record saved");

        document.getElementById("name").value = "";
        document.getElementById("age").value = "";
        document.getElementById("location").value = "";
        document.getElementById("condition").value = "";
      }

      function displayRecords() {
        let container = document.getElementById("records");
        container.innerHTML = "";

        records.forEach((r, index) => {
          let div = document.createElement("div");
          div.className = "record";

          div.innerHTML = `
<button class="delete" onclick="deleteRecord(${index})">Delete</button>
<b>Name:</b> ${r.name}<br>
<b>Age:</b> ${r.age}<br>
<b>Location:</b> ${r.location}<br>
<b>Health Condition:</b> ${r.condition}
`;

          container.appendChild(div);
        });
      }

      function deleteRecord(i) {
        records.splice(i, 1);
        saveStorage();
        displayRecords();
      }

      function searchRecord(text) {
        text = text.toLowerCase();

        let filtered = records.filter(
          (r) =>
            r.name.toLowerCase().includes(text) ||
            r.age.toString().includes(text) ||
            r.location.toLowerCase().includes(text) ||
            r.condition.toLowerCase().includes(text),
        );

        let container = document.getElementById("records");
        container.innerHTML = "";

        filtered.forEach((r, index) => {
          let div = document.createElement("div");
          div.className = "record";

          div.innerHTML = `
<button class="delete" onclick="deleteRecord(${index})">Delete</button>
<b>Name:</b> ${r.name}<br>
<b>Age:</b> ${r.age}<br>
<b>Location:</b> ${r.location}<br>
<b>Health Condition:</b> ${r.condition}
`;

          container.appendChild(div);
        });
      }

      function openRecordsPage() {
        document.getElementById("registerPage").classList.add("hidden");
        document.getElementById("recordsPage").classList.remove("hidden");
        displayRecords();
      }

      function openRegisterPage() {
        document.getElementById("recordsPage").classList.add("hidden");
        document.getElementById("registerPage").classList.remove("hidden");
      }
    </script>

    <h2 style="color: #fff176; text-align: center; margin-bottom: 20px">
      Challenges Faced by Migrant Workers
    </h2>

    <p>
      Migrant workers in Kerala often travel long distances in search of
      employment in construction, manufacturing, and service industries. Many of
      these workers face difficult living conditions, limited access to
      healthcare facilities, and language barriers that make communication with
      doctors difficult. Because they frequently move between workplaces and
      regions, their medical records are usually kept on paper or not recorded
      at all. As a result, important health information such as previous
      illnesses, allergies, or treatments can easily be lost.
    </p>

    <p>
      This lack of accessible medical records creates serious problems when
      workers fall ill or experience accidents. Doctors may not have access to
      the worker's medical history, which can delay treatment and increase
      health risks. In many cases, migrant workers also struggle to receive
      consistent medical care because their health information is scattered or
      unavailable across different hospitals and clinics.
    </p>

    <h2 style="color: #fff176; text-align: center; margin: 30px 0 20px">
      Digital Solution for Healthcare Access
    </h2>

    <p>
      The Digital Health Record Management System provides a simple and
      accessible platform where migrant workers’ health information can be
      securely stored and managed. Through this website, healthcare providers
      can record essential medical details such as personal information, health
      conditions, and treatment history. This ensures that the worker’s health
      data is available whenever it is needed, regardless of where the worker
      moves for employment.
    </p>

    <p>
      By digitizing health records, this system supports faster medical
      diagnosis, better continuity of care, and improved communication between
      workers and healthcare professionals. The platform also contributes to the
      goals of global healthcare improvement by supporting initiatives such as
      Sustainable Development Goal 3, which focuses on ensuring healthy lives
      and promoting well-being for all people.
    </p>
